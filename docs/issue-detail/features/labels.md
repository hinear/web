# Labels Feature Spec

## 목적

사용자가 이슈를 분류하고 검색 가능하게 만들기 위해 라벨을 추가하고 제거할 수 있게 한다.

## 요구사항

- 하나 이상의 라벨을 추가할 수 있다.
- 기존 라벨을 제거할 수 있다.
- 중복 라벨은 허용하지 않는다.
- 라벨 추가/삭제 성공 시 activity log에 남는다.
- 저장 실패 시 이전 라벨 집합으로 롤백한다.

## UX 원칙

- 현재 라벨 목록은 한눈에 보여야 한다.
- 라벨 추가와 제거는 과도한 이동 없이 가능해야 한다.

## 테스트 포인트

- 기존 라벨이 렌더링된다
- 라벨을 추가할 수 있다
- 라벨을 제거할 수 있다
- 중복 라벨 추가가 방지된다
- 실패 시 이전 라벨 집합으로 롤백된다
- label 변경 activity log가 표시된다
