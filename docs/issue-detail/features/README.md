# Issue Detail Features

이 디렉터리는 `issue detail` 화면을 피처 단위로 쪼갠 세부 스펙을 담는다.

상위 문서는 범위와 아키텍처를 설명하고, 여기 문서는 실제 구현 단위의 요구사항과 테스트 포인트를 정의한다.

## 문서 목록

- `title.md`
- `status.md`
- `priority.md`
- `assignee.md`
- `labels.md`
- `description.md`
- `comments.md`
- `activity-log.md`
- `notifications.md`
