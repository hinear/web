# Title Feature Spec

## 목적

사용자가 이슈 제목을 빠르게 읽고, 인라인으로 수정하고, 실패 시 안전하게 롤백할 수 있게 한다.

## 사용자 스토리

- 사용자는 상세 페이지 상단에서 현재 제목을 즉시 확인할 수 있다.
- 사용자는 페이지를 벗어나지 않고 제목을 수정할 수 있다.

## 요구사항

- 제목은 상단 영역에 표시된다.
- 제목은 인라인 편집 가능하다.
- 빈 문자열 또는 공백만 있는 제목은 저장할 수 없다.
- 저장 성공 시 최신 제목이 화면에 반영된다.
- 저장 실패 시 마지막 저장값으로 롤백한다.
- 성공한 제목 변경은 activity log에 남는다.

## 상태

- read mode
- edit mode
- saving
- save error

## 성공 기준

- 제목 수정 후 새로고침해도 반영된다.
- 저장 실패 후 잘못된 낙관적 상태가 남지 않는다.

## 테스트 포인트

- 기존 제목이 렌더링된다
- 제목을 수정하고 저장할 수 있다
- 빈 제목은 저장할 수 없다
- 저장 실패 시 이전 제목으로 돌아간다
- 제목 변경 activity log가 표시된다
